# Contributing to OpenClaw Browser Automation Skill

Thank you for your interest in contributing!

## How to Contribute

### Reporting Issues

1. Check existing issues to avoid duplicates
2. Describe the problem clearly with examples
3. Include OpenClaw and @playwright/mcp versions

### Submitting Improvements

1. Fork the repository
2. Edit `skills/browser-automation/SKILL.md`
3. Test with OpenClaw: `openclaw agent --message "test browser automation"`
4. Submit a pull request

## Skill Guidelines

- Be concise - instructions should be actionable
- Include examples for common patterns
- Follow OpenClaw skill best practices
